Wolk

Easy comparision of virtual machines in the cloud!

Comparing virtual machines offered by a cloud provider is quite frustrating. Using their own VM type descriptions or pricing pages and comparing them against the pages of other cloud providers is a boring, time consuming task that doesn't make anyone happy.

Wolk helps you to easily compare VMs in the cloud by providing you:

  • Accurate, up to date data on VMs offered by AWS, Microsoft Azure and Google Cloud Platform
  • A spreadsheet-like visualisation of the data
  • Easy to use filters to focus on what matters most for you
